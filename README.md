# EventManagement:
App to create events and register participants.
TDD Checklist:
路 Event creation works.
路 Registration stored in DB.
路 Prevent duplicate registration.
路 Show upcoming events only.

- C# / .NET for the backend
- React and Redux for the frontend
- xunit testing for backend and jest for frontend

How to Run

### Backend
1. Open the `EventManagementBackend` folder.
2. Restore and build the solution:
     ```bash
cd EventManagementBackend
dotnet restore
dotnet build
dotnet run

**###Frontend**
